What to optimize first

1. Immediate relief

The user should not have to think hard.

On /start, the first screen should do exactly three things:

* make the user feel seen
* make the next click obvious
* get them into a working tool in one tap

Best first-screen structure

Headline
Feel better in 60 seconds

Subhead
Choose what you need right now.

3 primary buttons

* Calm Me Down
* Help Me Think Clearly
* Understand This Feeling

Secondary tiny text
No pressure. No judgment. Start where you are.

That is stronger than a complex explanation-heavy landing.

⸻

2. Click-to-response speed

The first AI response should not feel like “chatbot talk.”
It should feel like guided relief.

For each first-click path

Calm Me Down

* default tool: box breathing or grounding_54321
* response shape: 1 validation line + 1 exercise intro + numbered steps

Help Me Think Clearly

* default tool: thought_reframe
* response shape: 1 reflection line + 4-step reframe

Understand This Feeling

* default tool: emotional_checkin
* response shape: name feeling → body location → need

The first response should be shorter than your average current reply.

Rule:

* first reply target: 80–160 words
* max 1 question at the end
* no long disclaimer blocks unless crisis path triggers

⸻

3. Repeat-use design

A user should leave with one simple next action.

At the end of successful non-crisis sessions, append one of these:

* Come back tomorrow for a 1-minute reset.
* Want to save today as part of your streak?
* Would you like one small next step for tonight?

That turns a chat into a habit loop.

⸻

The habit loop system

Best lightweight streak design

Do not overbuild gamification first.

Start with:

* 1 check-in per day
* 1 streak counter
* 1 “today’s reset completed” state
* 1 return prompt

Streak event definition

A day counts when user completes one of:

* AI chat with successful tool guidance
* direct exercise completion
* daily check-in
* mood check + reflection

Display copy

* Day 1: You showed up for yourself today.
* Day 3: You’re building a real pattern.
* Day 7: A week of showing up matters.

Avoid juvenile badges first.
Lead with meaning, not gimmicks.

Soft paywall strategy

Do not put the paywall before the first win.

That would hurt trust.

Correct paywall timing

Show upgrade only after the user has experienced value.

Best trigger windows:

* after 3–5 successful sessions
* after streak day 3
* when trying to open advanced personalization
* when requesting deeper guided programs

Pro
