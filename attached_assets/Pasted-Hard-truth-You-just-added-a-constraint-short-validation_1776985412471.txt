⸻

STEP 4 — Inject hint (not override)

In provider:

import { EMOTION_PHRASES } from "./emotionPhrases.mjs";

Then:

const emotionHint =
  inferredStates?.length
    ? EMOTION_PHRASES[inferredStates[0]]?.join(" | ")
    : null;

Add system message:

...(emotionHint
  ? [{
      role: "system",
      content:
        "Possible emotional tones (vary naturally): " +
        emotionHint
    }]
  : [])
