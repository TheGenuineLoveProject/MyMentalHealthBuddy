- scripts/print-routes-diff.mjs
- content/categoryOrder.js
- reports/ (generated artifacts)

GENERATOR OUTPUT ROOT (LOCKED):
client/src/pages/generated/

GENERATOR MODES (REQUIRED):
1) --mode=all

2) --mode=landing
   - generates ONLY Landing routes + aliases + dynamic stubs + 404
3) --mode=category --category="Wellness & Healing Tools"

4) --mode=range --from=A --to=C

- Generator MUST refuse to overwrite any existing page unless the file contains a line:

- Generator must also support --force to override (still must preserve a backup copy into reports/backups/).

ALIAS HANDLING (LOCKED CHOICE):
- Use canonical render (no redirect): generated alias files must import and re-export canonical page component, so SEO stays stable and UI consistent.

  blog-[param].jsx and community-discussion-[param].jsx

- /* => 404.jsx (or not-found.jsx if that's your system, but be consistent)
