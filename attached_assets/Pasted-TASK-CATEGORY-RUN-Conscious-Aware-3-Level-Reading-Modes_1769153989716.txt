TASK: CATEGORY RUN — Conscious-Aware 3-Level Reading Modes (Beginner / Standard / Deep)

INTENT:
Build a conscious-aware reading mode system that honors autonomy, emotional safety, accessibility, cultural humility, and nervous-system pacing — while staying technically strict, fast, and legally safer.

CORE PRINCIPLES (CONSCIOUS-AWARE DESIGN):
- Consent-led: user always chooses intensity/detail; default is gentle.
- Non-coercive: never push; always offer options (“you can”, “if you’d like”).
- Identity-safe: do not assume culture, gender, ability, belief, or background.
- Non-pathologizing: avoid labeling people; avoid “fixing you” language.
- Nervous-system respectful: slow motion, breathable spacing, no surprise intensity.
- Trauma-aware: include pause/stop cues; offer grounding exits; link crisis resources.
- Culturally humble: avoid universal claims; present practices as optional tools.
- Legally safer: educational support only; no diagnosis, no medical/treatment claims.

NON-NEGOTIABLE TECH:
- Next.js (pages router), React
- CSS Modules only
- Motion: AOS + GSAP
  - AOS MUST be initialized with once:true
  - GSAP MUST be disabled when prefers-reduced-motion: reduce
- Accessibility: semantic landmarks, skip link, aria labels, focus-visible, keyboard safe
- Performance: minimal DOM nesting, lazy load, no layout-thrashing animations
- Output must compile. NO TODOs. NO partial solutions.

CONSCIOUS-AWARE COPY RULES (MANDATORY):
- Autonomy language: “You can try…”, “If you’d like…”, “Take what helps…”
- Normalize pacing: “It’s okay to pause.” “Go at your pace.”
- Avoid prescriptive “should” and avoid moral framing.
- Avoid pathologizing terms and identity assumptions.
- Avoid spiritual absolutes; if spiritual tone exists, keep it optional and inclusive.
- For practice pages, include a “Pause/Stop” line:
  - “If this feels activating, pause and try something simpler.”
- Provide gentle safety routing:
  - Always include link to /crisis and a short supportive line like:
    “If you’re in immediate danger or feel unsafe, seek local emergency help.”

LEGAL SAFETY RULES (STRICT):
- No diagnosis, no medical advice, no treatment claims, no guaranteed outcomes.
- Avoid “clinically proven to cure/repair”.
- Use cautious phrasing: “may help”, “some people find”, “often used for”.
- Add global disclaimer line (footer or layout):
  “Educational support — not medical advice.”

BEGINNER / STANDARD / DEEP GUIDELINES:
- Beginner: plain language, short sentences, define terms briefly, kind tone.
- Standard: warm + grounded, structured bullets, clear next step CTA.
- Deep: more context, optional “Why this might help” and “When to pause”.
Always keep paragraphs short and breathable.
