TASK: Create a robust tiny formatter script: /scripts/print-routes-diff.mjs that reads reports/routes.diff.json and prints a PR-ready Markdown comment to stdout.

REQUIREMENTS:
1) Create file: /scripts/print-routes-diff.mjs
   - Node built-ins only (fs/path).
   - Input:
     - Default path: reports/routes.diff.json
     - Optional CLI arg: --file=<path> (override)
     - Optional CLI arg: --max=<n> (limit items per section; default 10)
   - Behavior:
     - If diff file does not exist:
       - Print: "✅ Routes manifest clean (no reports/routes.diff.json found)."
       - Exit 0.
     - If diff exists and status === "clean":
       - Print a short clean message and exit 0.
     - If status === "drift":
       - Print a Markdown block suitable for a PR comment, including:
         - Title line: "## Route Manifest Drift Detected"
         - Summary bullets with counts:
           Added / Removed / Title Changed / Category Changed / Other Changes (hash)
         - Recommended command line in a code fence (if present)
         - Sections for each list, with route lines:
           - Added: `+ /route (Category)`
           - Removed: `- /route (Category)`
           - Title changed: `~ /route: "Old" → "New"`
           - Category changed: `~ /route: "Old Category" → "New Category"`
           - Other changes: `~ /route (hash changed)`
         - Truncate each section to --max, then show “…and N more”
       - Exit code:
         - Exit 2 on drift (so CI can treat it distinctly).

2) Add package.json scripts:
   - "print:routes:diff": "node scripts/print-routes-diff.mjs"
   - "print:routes:diff:ci": "node scripts/print-routes-diff.mjs --max=20"
   (Only add if missing; keep existing scripts.)

3) Keep output stable:
   - Sort routes in each section by route.
   - Ensure no noisy whitespace.

OUTPUT REQUIREMENTS:
- Provide FULL new /scripts/print-routes-diff.mjs
- Provide FULL updated package.json (only if modified)
- Show a short example of the produced Markdown (a few lines)

DONE GATE:
✅ If routes.diff.json is missing => script prints clean message and exits 0
✅ If routes.diff.json status=drift => script prints Markdown + exits 2
✅ Output respects --max and --file flags
FAIL THE RUN if any DONE GATE item is not satisfied.
