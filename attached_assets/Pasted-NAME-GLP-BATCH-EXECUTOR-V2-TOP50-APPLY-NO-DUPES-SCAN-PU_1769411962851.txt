1) IN-DEPTH NO-DUPES SCAN (MANDATORY BEFORE EVERY PATCH)
========================================================
Run ALL scans and write findings into /docs/NO_DUPES_REPORT.md (append with timestamp).

A) Repo + build + env sanity
- git status || true
- node -v || true
- npm -v || true
- cat package.json
- ls -la

B) Duplicate folder patterns (backup/copy/old/tmp)
- find . -maxdepth 6 -type d \( -iname "*backup*" -o -iname "*copy*" -o -iname "*old*" -o -iname "*tmp*" -o -iname "*archive*" \)

C) Duplicate package manifests / monorepo collisions
- find . -maxdepth 6 -name "package.json" -print
- find . -maxdepth 6 -name "package-lock.json" -print
- find . -maxdepth 6 -name "pnpm-lock.yaml" -print
- find . -maxdepth 6 -name "yarn.lock" -print

D) Placeholder/unfinished code + repeated prompts
- grep -R "TODO\|FIXME\|HACK\|placeholder\|BEGIN PASTE\|END PASTE" -n client server scripts . || true

E) Duplicate feature risk scan (auth/billing/db/openai/routes)
- grep -R "jsonwebtoken\|passport\|lucia\|next-auth\|auth" -n server client || true
- grep -R "stripe\|webhook\|checkout\|billing" -n server client || true
- grep -R "drizzle\|schema\.ts\|migrate\|neon\|postgres" -n server || true
- grep -R "openai\|gpt\|chat" -n server client || true
- grep -R "routeKey\|canonicalPath\|internalLinks" -n client/src || true

F) Dead routes / duplicated routes scan
- grep -R "createBrowserRouter\|react-router\|Routes\|Route" -n client/src || true
- grep -R "/admin\|/tools\|/crisis" -n client/src server || true

OUTPUT REQUIRED (in /docs/NO_DUPES_REPORT.md):
1) Duplicate folders list + recommended handling (keep/ignore/merge later)
2) Duplicate manifests list + collision risks
3) Duplicate feature candidates (e.g., 2 stripe configs, 2 auth flows)
4) “Already implemented” list (skip work)
5) “Blockers” list (must fix first)

STOP and produce a DRY-RUN plan for Patch 1.
