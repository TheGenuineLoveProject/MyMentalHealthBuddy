2) Add a verification script safely

Run this first to inspect package.json:

cat package.json

If package.json already has a scripts block with build and test or test:e2e, use one of these edits.

Option A — if you already have test

python - <<'PY'
import json
from pathlib import Path

p = Path("package.json")
data = json.loads(p.read_text())
scripts = data.setdefault("scripts", {})
scripts["verify:system"] = "npm run build && npm run test"
p.write_text(json.dumps(data, indent=2) + "\n")
print("Added scripts.verify:system = npm run build && npm run test")
PY

Option B — if you have test:e2e but not test

python - <<'PY'
import json
from pathlib import Path

p = Path("package.json")
data = json.loads(p.read_text())

p.write_text(json.dumps(data, indent=2) + "\n")

PY
