🔁 PHASE 2 — Return-after-drop system (critical retention)

Goal

Recover users who disappear for 2–3 days

⸻

STEP 3 — Backend: detect inactivity

Edit: server/routes/streaks.mjs

Add:

const now = new Date();
const last = user.lastCheckin ? new Date(user.lastCheckin) : null;
let daysAway = 0;
if (last) {
  daysAway = Math.floor((now - last) / (1000 * 60 * 60 * 24));
}

Include in response:

return res.json({
  streak,
  daysAway
});
