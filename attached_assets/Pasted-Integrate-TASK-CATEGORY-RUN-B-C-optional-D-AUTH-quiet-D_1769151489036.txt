CATEGORY-SPECIFIC STIMULATION RULES (NON-NEGOTIABLE):

(1) AUTH = QUIET / MINIMAL / SAFE  (tone:"quiet")
- Visual tone: quiet, minimal, safe. Clarity > flourish.
- Background: sacred-pattern opacity extremely low; aura gradient very subtle OR disabled on auth pages.
- Motion: minimal. AOS allowed for gentle entrance only. No looping glows near forms. No shimmer. No parallax.
- Layout: centered form card, max-width ~420–520px, generous padding, high contrast.
- CTAs: 1 primary action + 1 secondary link only.
- Copy: short sentences. No pressure. Add 1 reassurance line per page (“You can go slowly. You’re safe here.”)
- Error states: gentle, specific, actionable (no shame language).

(2) DASHBOARD = STRUCTURED / LOW COGNITIVE LOAD (tone:"structured")
- Visual tone: calm, organized, “one clear next step.”
- Background: pattern + aura allowed but restrained.
- Motion: purposeful entrances only (AOS once:true). GSAP micro only (opacity/transform).
- Layout: scannable grid, consistent spacing, no dense walls of text.
- Above the fold must include:
  A) Title + one-sentence purpose
  B) “Next best step” primary CTA (single)
  C) 2–4 small status cards/chips (optional)
- Below the fold: “Today’s focus” + “Recent/Saved” (if applicable) + “Guided options” (<= 3 cards)

(3) WELLNESS = PRACTICE-FIRST / SOMATIC / SHORT ACTIONS (tone:"practice")
- Visual tone: soothing, embodied, sensory-friendly. The practice is the product.
- Background: pattern + aura allowed but SOFT; no sharp contrast patterns; avoid visual noise.
- Motion: “breath-paced” and optional. AOS once:true. GSAP micro only. No fast transitions. No flash.
- Layout rules:
  - Put the practice UI above the fold (a single “Start” action).
  - Default to 1–3 minute options.
  - Provide “Stop / Pause / Reset” affordance in copy and/or UI.
  - Content chunking: short paragraphs + bullets; no long sections.
- Copy rules:
  - Begin with body cue + permission (“If you can, soften your shoulders… If not, that’s okay.”)
  - Use neutral, non-triggering language.
  - Avoid absolutes (“always/never”), avoid pressure (“must/should”).
- Add a “Safety exit” link on every wellness page:
  - label: “Need immediate support?” href: “/crisis”
- Add three-tier content (Beginner / Intermediate / Advanced) but practice-oriented:
  - Beginner: 1 tiny step (10–30 seconds)
  - Intermediate: 1–3 minutes
  - Advanced: 3–10 minutes, reflection prompt optional
