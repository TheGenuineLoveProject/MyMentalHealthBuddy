# Visible Webpage Readiness Checklist

## Before Runtime Page Build
Confirm:
- route exists
- page renders 200
- copy is approved
- disclaimer exists
- CTA exists
- price is clear
- no clinical claims
- no broken imports
- npm run verify:all passes
- npm run build passes

## First Visible Page To Build
7-Day Self-Understanding Reset landing page.

## Rule
One visible page at a time. Verify before adding checkout or billing.
