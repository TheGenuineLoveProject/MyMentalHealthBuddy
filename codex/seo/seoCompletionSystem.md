# SEO Completion System

## Required SEO For Every Public Page
- clear title
- meta description
- canonical link
- Open Graph title
- Open Graph description
- Twitter metadata
- user-intent keyword
- ethical CTA
- disclaimer where wellness claims appear

## Priority Pages
1. /7-day-reset
2. /pricing
3. /features
4. /about
5. /blog
6. /launch

## SEO Positioning
Sell practical self-understanding, emotional clarity, nervous-system calm, guided reflection, and personal growth.

## Do Not Claim
- cure
- diagnosis
- therapy replacement
- guaranteed healing
- emergency support
