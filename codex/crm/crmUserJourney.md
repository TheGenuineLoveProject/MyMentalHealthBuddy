# CRM User Journey

## User Stages
1. Visitor
2. Free reflection user
3. Email subscriber
4. 7-Day Reset customer
5. Monthly subscriber
6. Long-term growth member

## CRM Goals
- build trust
- educate gently
- encourage reflection
- invite growth
- never pressure
- never shame
- never exploit pain

## Email Flow
1. Welcome
2. Emotional clarity prompt
3. Self-awareness education
4. Nervous-system calming practice
5. Invitation to 7-Day Reset
6. Weekly growth reflection

## Compliance Boundary
CRM cannot use crisis, journal, private healing, provider-review, or sensitive emotional disclosures for sales targeting.
