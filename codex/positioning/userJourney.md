# User Journey

## Stage 1: Arrival
User feels overwhelmed, curious, stuck, or ready to grow.

## Stage 2: Safety
User understands the platform is supportive, private, nonjudgmental, and not manipulative.

## Stage 3: Reflection
User identifies emotions, patterns, needs, values, and goals.

## Stage 4: Practice
User uses tools for grounding, journaling, regulation, self-compassion, and clarity.

## Stage 5: Proof
User sees measurable growth signals over time.

## Stage 6: Integration
User becomes more authentic, regulated, aware, and aligned in daily life.
