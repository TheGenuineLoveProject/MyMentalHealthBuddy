# Emotionally Intelligent Automation

## Purpose
Automation should support clarity, calm, compassion, self-awareness, and user empowerment.

## Allowed Automation
- reminders chosen by the user
- reflection prompts
- progress summaries
- wellbeing check-ins
- educational guidance
- nonjudgmental motivation

## Forbidden Automation
- emotional pressure
- addiction loops
- fear-based urgency
- crisis monetization
- hidden persuasion
- vulnerability scoring
- manipulative retention
