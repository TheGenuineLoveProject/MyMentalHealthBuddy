# Publishing Style Guide — The Genuine Love Project

## Voice & Tone

- **Warm but not saccharine.** Genuine, not performative.
- **Gentle but honest.** We don't sugarcoat, but we don't wound.
- **Optional always.** Everything is an invitation, never a demand.
- **Trauma-informed.** Assume the reader may be in a vulnerable state.

## Language Rules

### Always Use
- "If this feels right…"
- "You can ignore this entirely."
- "At your own pace."
- "No pressure."
- "You're welcome here."
- "This is optional."

### Never Use
- Urgency: "Don't miss," "Last chance," "Today only," "Act now," "Hurry"
- Shame: "What's wrong with you," "You should be ashamed"
- Guilt: "You must," "Stop making excuses"
- Medical: "Cure," "Diagnose," "Prescribe," "Guaranteed healing," "Will heal you"
- Dependency: "You need us," "You can't do this without"

## CTA Rules

- One CTA per email maximum
- Gentle framing: "If you'd like to explore…" not "Click now!"
- Allowed CTA targets: /tools, /journal, /reflection, /blog, /pricing, /newsletter, /crisis, /about
- Always include crisis resources in footer

## Formatting

- Short paragraphs (2-3 sentences max)
- Use headers to break up content
- Include breathing room (whitespace)
- Mobile-first: test on small screens

## Safety

- Every email must include the crisis-safe footer
- Every email must include an unsubscribe mechanism
- Never reference private user data in newsletters
- Never automate sending — human approval required
