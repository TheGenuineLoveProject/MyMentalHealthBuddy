# 5 Launch Blog Ideas

Short briefs for the first five posts. Full drafts of the same ideas live in `5_LAUNCH_BLOG_DRAFTS.md`. Use this file when you need the one-screen pitch; use the drafts file when you're ready to publish.

Theme: **Feel better in 60 seconds.**
Default CTA on every post: a single soft link to **/start**.

---

## 1. Why Your Anxiety Spikes When Life Feels Uncertain
- **Module / tool**: `anxiety` → `box_breathing`
- **Hook**: When you can't calm down, stop trying to.
- **Truth**: Anxiety is a prediction system, not a personal failure.
- **Tool**: A 60-second box breath (four in, four hold, four out, four hold).
- **CTA**: Try the **Calm Me Down** flow → /start

## 2. How to Stop a Thought Loop Without Fighting Your Mind
- **Module / tool**: `thought_pattern` → `thought_reframe`
- **Hook**: A thought is a draft, not a verdict.
- **Truth**: Arguing with every thought makes the loop louder.
- **Tool**: A four-step reframe — catch, underline absolutes, write the truer version, notice the body.
- **CTA**: Try **Help Me Think Clearly** → /start

## 3. What Emotional Overwhelm Is Trying to Tell You
- **Module / tool**: `emotional_processing` → `emotional_checkin`
- **Hook**: Name it, then locate it.
- **Truth**: Feelings live in the body. Skipping the body keeps you in your head.
- **Tool**: Three lines — *I feel ___. It lives in my ___. It feels like ___.*
- **CTA**: Try **Understand This Feeling** → /start

## 4. Why You Pull Away When You Need Support
- **Module / tool**: `relationship` → `relationship_repair`
- **Hook**: The repair is the relationship.
- **Truth**: Stress responses can look like distance. The next move is showing up, not explaining.
- **Tool**: A four-line repair script (thinking about us / what I want to be true / what I want to own / how it landed).
- **CTA**: Try a guided **Relationship Repair** → /start

## 5. The 1-Minute Reset: A Simple Daily Habit for Self-Regulation
- **Module / tool**: `self_regulation` → `overload_reset` (paired with `box_breathing`)
- **Hook**: Calm is a skill, not a mood.
- **Truth**: Small daily resets compound into a regulated nervous system.
- **Tool**: One 60-second reset per day. Streaks build self-trust.
- **CTA**: Start your **1-minute reset** → /start
