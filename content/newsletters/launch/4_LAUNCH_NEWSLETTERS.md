# 4 Launch Newsletters

Concise outlines for the first month. Full sendable copy lives in `4_WEEK_NEWSLETTER_STARTER.md`. Use this file when planning the calendar; use the starter file when you're ready to send.

Theme: **Feel better in 60 seconds.**
Send window: Tuesday or Wednesday morning. Hard limit: 350 words per issue.

---

## Week 1 — Calm First
- **Subject**: Start with one breath
- **Module / tool**: `anxiety` → `box_breathing` (Calm Me Down)
- **Hook**: When you can't calm down, stop trying to.
- **Truth**: You don't need to solve everything before you regulate.
- **Tool**: One round of box breathing — 16 seconds, then decide.
- **Reflection**: What feels most urgent right now?
- **CTA**: Try a 60-second reset → /start

## Week 2 — Thought Loops
- **Subject**: When your mind keeps replaying
- **Module / tool**: `thought_pattern` → `thought_reframe` (Help Me Think Clearly)
- **Hook**: A thought is a draft, not a verdict.
- **Truth**: Repeating thoughts often need structure, not shame.
- **Tool**: A four-step reframe — catch, underline absolutes, write the truer version, notice the body.
- **Reflection**: What thought keeps returning this week?
- **CTA**: Try the guided reframe → /start

## Week 3 — Emotional Clarity
- **Subject**: Name it, then locate it
- **Module / tool**: `emotional_processing` → `emotional_checkin` (Understand This Feeling)
- **Hook**: Feelings live in the body, not the head.
- **Truth**: Naming and locating, together, do most of the work.
- **Tool**: Three lines — *I feel ___. It lives in my ___. It feels like ___.*
- **Reflection**: What feeling needs attention right now?
- **CTA**: Try a one-screen check-in → /start

## Week 4 — Patterns
- **Subject**: The pattern is not the person
- **Module / tool**: `loop_detection` → `pattern_interrupt`
- **Hook**: The loop isn't proof you're stuck.
- **Truth**: Repeating behavior is information, not identity.
- **Tool**: Stand up. Ask: *what was I actually looking for?* Pick a smaller version of the answer.
- **Reflection**: What happens right before the loop begins?
- **CTA**: Try a guided interrupt → /start
