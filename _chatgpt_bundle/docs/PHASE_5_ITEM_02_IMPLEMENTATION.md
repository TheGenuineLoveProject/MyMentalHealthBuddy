# Phase 5 — Item 02 Implementation
Improve UI Spacing & Visual Calm

STATUS: ACTIVE (Unlocked for this item only)

---

## OBJECTIVE

Reduce cognitive load and visual stress across the UI
by improving spacing, margins, and layout rhythm.

This is a **design-quality improvement**, not a feature change.

---

## SCOPE

Applies to:
• Reflection screens
• Journaling inputs
• Completion / closing screens
• Any text-dense UI areas

Does NOT apply to:
• Navigation structure
• Feature additions
• Content changes

---

## SPACING PRINCIPLES

• White space is intentional
• Fewer elements per screen
• One primary focus per view
• Calm vertical rhythm

---

## BASELINE RULES

• Minimum vertical spacing between sections: 24–32px
• Line height: 1.4–1.6
• No stacked text blocks without space
• Buttons never touch text blocks

---

## BEFORE / AFTER EXAMPLE

Before:
Text blocks stacked tightly, dense feeling.

After:
Clear separation, slower reading pace, softer entry.

---

## VALIDATION CHECKLIST

Before marking complete:
[ ] Can the screen be scanned calmly?
[ ] Is the eye drawn to only one main element?
[ ] Does it feel slower, not busier?

---

## COMPLETION CRITERIA

✔ All reflection views updated  
✔ Journaling input feels spacious  
✔ Completion screens feel gentle  

---

## RESULT

The interface supports calm
instead of competing for attention.
