# Phase 5 — Item 03 Implementation
Improve Completion / Closure Experience

STATUS: ACTIVE (Unlocked for Item 03 only)

---

## OBJECTIVE

Strengthen the end-of-reflection experience so users
feel grounded, complete, and free to stop—without pressure
to continue, share, or engage further.

---

## SCOPE

Applies to:
• Reflection completion screens
• Journaling completion state
• Exit moments from reflection flows

Does NOT apply to:
• New features
• Notifications
• Engagement mechanics
• Content expansion

---

## CLOSURE PRINCIPLES

• Completion is a resting point, not a funnel
• No next-step pressure
• No metrics
• Calm emotional landing

---

## REQUIRED ELEMENTS (ONE OR MORE)

• Gentle closing line (one sentence max)
• Optional grounding cue (e.g., breathe, pause)
• Clear exit option

---

## APPROVED CLOSING COPY (EXAMPLES)

• “You’ve done enough for now.”
• “You can return whenever it feels right.”
• “It’s okay to rest after reflecting.”

---

## UI RULES

• No primary CTA
• Secondary actions must be optional and soft
• Generous spacing
• Calm typography

---

## VALIDATION CHECKLIST

Before marking complete:
[ ] Does the user feel finished?
[ ] Is there zero pressure to continue?
[ ] Is rest implicitly allowed?

---

## COMPLETION CRITERIA

✔ Closure copy added
✔ Exit is clear and gentle
✔ No engagement pressure remains

---

## RESULT

The experience ends with dignity,
not demand.
