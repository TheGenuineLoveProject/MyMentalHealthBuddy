# Canva Brand Kit - The Genuine Love Project

> These templates are the ONLY approved designs for social publishing.

## Rules
- Colors must come from the brand tokens below
- Fonts must follow the typography guide
- Text may change; layout and colors may NOT
- Templates are duplicated, never edited directly

## Purpose
Enable calm, brand-safe, beginner-friendly publishing without design decisions each time.

---

## Brand Kit Setup

### Colors (Copy to Canva Brand Kit)
Add these exact hex codes to your Canva Brand Kit:

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Primary (Sage Green) | `#6D9B8D` | Headlines, buttons, key elements |
| Secondary (Mint) | `#A4C3B2` | Backgrounds, cards, accents |
| Accent (Warm Blush) | `#EAC3B5` | Highlights, callouts, warmth |
| Gold (Achievement) | `#EAC33B` | Premium elements, badges |
| Dark Text | `#1C1C1C` | Body text, paragraphs |
| Light Background | `#F6F7F8` | Page backgrounds |

### Fonts

**Display/Headlines:**
- Font: **Playfair Display**
- Weight: Bold (700) or SemiBold (600)
- Use for: Titles, headlines, hero text

**Body Text:**
- Font: **Inter**
- Weight: Regular (400) or Medium (500)
- Use for: Paragraphs, descriptions, captions

**If fonts unavailable in Canva:**
- Display alternative: **Bodoni Moda** or **Cormorant**
- Body alternative: **Lato** or **Open Sans**

## Spacing Rules
| Size | Pixels | Use Case |
|------|--------|----------|
| Tight | 8px | Between related elements |
| Normal | 16px | Standard spacing |
| Comfortable | 24px | Section padding |
| Generous | 32-48px | Hero sections, CTAs |

## Border Radius
- Cards: 16px
- Buttons: 8px
- Pills/Tags: Full rounded
- Images: 8-16px
