# Phase 5 — Item 01 Implementation
Improve Reflection Prompt Clarity

STATUS: ACTIVE

---

## OBJECTIVE

Improve clarity, softness, and accessibility of reflection prompts
without changing meaning or adding pressure.

---

## SCOPE

Applies to:
• Reflection prompts
• Compassion statements
• Educational microcopy

Does NOT apply to:
• UX flow changes
• New features
• Automation logic

---

## CLARITY RULES

Each reflection prompt must:
• Be one sentence
• Use plain language
• Avoid metaphors that confuse
• Avoid “why” questions
• Invite, not instruct

---

## BEFORE / AFTER EXAMPLE

Before:
“What patterns are emerging in your internal landscape today?”

After:
“What have you been noticing about how you feel today?”

---

## APPROVED LANGUAGE SHIFTS

Replace:
• “explore” → “notice”
• “process” → “experience”
• “examine” → “observe”
• “should” → remove entirely

---

## VALIDATION CHECKLIST

Before approving an updated prompt:
[ ] Would this feel safe to read when overwhelmed?
[ ] Can it be understood in one pass?
[ ] Does it allow choice?

---

## COMPLETION CRITERIA

✔ All prompts reviewed  
✔ Language simplified  
✔ Tone remains gentle  

---

## RESULT

Reflection content is clearer,
more accessible,
and emotionally safer.
