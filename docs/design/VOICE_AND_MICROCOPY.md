# Voice & Microcopy - The Genuine Love Project

## Brand Voice
- **Warm** - Caring, supportive, genuine
- **Grounded** - Calm, present, stable
- **Compassionate** - Understanding, non-judgmental
- **Empowering** - Encouraging growth, not dependency

## Tone Guidelines
| Context | Tone |
|---------|------|
| Onboarding | Welcoming, reassuring |
| Journaling | Reflective, supportive |
| Crisis | Calm, immediate, resourceful |
| Achievements | Celebratory, encouraging |
| Errors | Helpful, solution-focused |

## Button Labels

### Do Use
- "Get Started" (not "Sign Up Now!")
- "Continue" (not "Next Step >>")
- "Save Changes" (not "Submit")
- "Talk to Companion" (not "Chat with AI")

### Avoid
- Urgency language ("Act Now!", "Limited Time!")
- Clinical terms ("Therapy", "Treatment", "Diagnosis")
- Overpromises ("Cure", "Fix", "Heal You")

## Empty States

```
No journal entries yet.
This is a safe space to reflect on your thoughts and feelings.
[Start Writing]
```

## Microcopy Principles
1. **Be human** - Write like you're talking to a friend
2. **Be brief** - Respect attention and cognitive load
3. **Be helpful** - Every word should serve the user
4. **Be honest** - No manipulative patterns
5. **Be inclusive** - Avoid assumptions about experience
