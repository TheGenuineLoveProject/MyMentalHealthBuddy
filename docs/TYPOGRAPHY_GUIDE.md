# The Genuine Love Project — Typography Guide (v17)
**Intent:** Soft authority + calm clarity.  
**Primary fonts:** Playfair Display (headings) + Inter (UI/body).

## 1) Font Pair
- **Headings (Serif):** Playfair Display
- **Body/UI (Sans):** Inter
- **Fallbacks:** serif: Georgia; sans: system-ui, -apple-system, Segoe UI

## 2) Type Scale (Aligned to Canva)
- **Title / H1:** Playfair Display 56 (mobile: 40–44)
- **Subtitle / H2:** Playfair Display 36
- **Heading / H3:** Playfair Display 32
- **Subheading / Lead:** Inter 28
- **Section Header:** Playfair Display 24
- **Body:** Inter 18
- **Quote:** Inter 20 (italic allowed)
- **Caption:** Inter 14

## 3) Weights (Minimize Variety)
- Inter: Regular (400), Medium (500), Bold (700)
- Playfair: Regular (400), SemiBold (600), Bold (700)
**Rule:** Don’t mix more than 3 weights on one page.

## 4) Typesetting Rules (Hard Rules)
- One H1 per page.
- Body line-height: 1.60–1.75 (default 1.65).
- Headings line-height: 1.15–1.25.
- Reading line length: 60–75 characters.
- Avoid ALL CAPS in paragraphs.
- No stretched/warped text.

## 5) Voice + Typography Alignment
- Headings: poetic but clear
- Body: simple, grounded, non-clinical
- Buttons: gentle imperatives (“Begin quietly”, “Try this”, “Explore”)

## 6) UI Microcopy Standards
- Errors: calm + actionable (“That didn’t work yet. Try again.”)
- Empty states: supportive (“Nothing here yet—when you’re ready, start small.”)
- Disclaimers: always visible near tools (non-therapy, non-crisis)
