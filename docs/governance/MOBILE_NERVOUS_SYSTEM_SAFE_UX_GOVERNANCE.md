# Mobile Nervous-System-Safe UX Governance

## Purpose
Ensure the mobile experience is calm, readable, emotionally safe, thumb-friendly, accessible, and low-friction.

## Core Law
Mobile UX must reduce emotional load, not increase it.

## Audit Areas
- thumb reach
- font readability
- spacing
- tap targets
- reduced motion
- calming transitions
- low cognitive density
- crisis visibility
- privacy visibility
- tool discoverability
- journey continuity
- page speed

## Forbidden Patterns
- tiny tap targets
- dense text walls
- flashing motion
- urgency pressure
- hidden crisis resources
- confusing navigation
- excessive modal stacks
- manipulative streaks
- emotional dependency loops
