# Canonical Ownership Rules

## Runtime Truth
- Only active runtime directories may control production behavior.

## No Duplicate Ownership
- One route owner
- One state owner
- One design owner
- One provider owner
- One orchestration owner

## Ignored Systems
- production-backups
- archives
- snapshots
- static-export
- temp

## Governance Principle
Every system must have:
- canonical ownership
- deterministic responsibility
- observable behavior
- rollback safety
