# Unified Wellness Dashboard Canonization

## Purpose
Define one canonical dashboard system so the platform does not fragment into duplicate dashboards, duplicate wellness hubs, or competing command centers.

## Canonical Dashboard Role
The dashboard is the calm user command center for:
- continue your journey
- saved tools
- mood awareness
- reflections
- guided journeys
- calm room access
- trust/safety links
- account/data controls

## Forbidden
- duplicate dashboard systems
- dashboard monetization pressure
- crisis content mixed with upsells
- hidden personalization
- emotional vulnerability scoring
- business logic inside healing flows

## Dashboard UX Law
The dashboard must reduce cognitive load, not increase it.
