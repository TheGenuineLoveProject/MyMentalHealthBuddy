# Content Intelligence Taxonomy Governance

## Purpose
Create a governed content taxonomy so every article, tool, journey, page, and resource belongs to a clear emotional, educational, and discovery pathway.

## Core Law
No random content. Every content item must support:
- user understanding
- emotional safety
- discoverability
- journey continuity
- trust
- non-clinical wellness education

## Primary Content Pillars
1. Emotional Awareness
2. Nervous System Regulation
3. Journaling + Reflection
4. Calm + Grounding
5. Sleep + Restoration
6. Self-Worth + Identity
7. Relationships + Connection
8. Burnout + Stress
9. Anxiety Support
10. Growth + Meaning

## Required Metadata
Every content/tool entry should eventually include:
- title
- slug
- contentType
- emotionalState
- journeyStage
- pillar
- relatedTools
- relatedJourneys
- safetyLevel
- clinicalBoundary
- seoIntent
- internalLinks

## Forbidden Content Patterns
- diagnosis claims
- cure claims
- shame language
- urgency manipulation
- emotional dependency framing
- business upsells inside healing content
- hidden targeting based on emotional vulnerability
