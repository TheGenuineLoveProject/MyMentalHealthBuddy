# Onboarding Flow

## Overview

The Genuine Love Project uses a calm, guided, step-by-step onboarding process designed to welcome users gently into their wellness journey.

## Principles

1. **Calm Introduction** - No overwhelming information dumps
2. **Consent-First** - Clear opt-in for all features
3. **Tier Selection** - Choose Beginner, Intermediate, or Advanced
4. **Goal Setting** - Define personal wellness goals
5. **Safety First** - Crisis resources always visible

## Flow Steps

### Step 1: Welcome
- Warm greeting
- Platform purpose explanation
- 18+ age confirmation

### Step 2: Account Setup
- Email/password or OAuth
- Privacy policy acknowledgment

### Step 3: Personalization
- Select content tier (Beginner/Intermediate/Advanced)
- Choose focus areas
- Set notification preferences

### Step 4: First Experience
- Guided tour of key features
- First journal prompt (optional)
- Crisis resources introduction
