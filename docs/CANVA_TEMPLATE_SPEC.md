Canva Template Spec Pack — matches site (Teal + Gold priority)

Global rules (all templates)
  •	Primary: Deep Teal #2F5D5D (headlines, CTAs, key UI color)
  •	Accent: Gold #EAC33B (underlines, bullets, badges, rings) — never body text
  •	Background: Ivory #FAF9F7 (default)
  •	Text: Charcoal #3A3A3A (all body + long copy)
  •	Supporting (sparingly, ≤15% of canvas): Sage #8FBF9F, Blossom #F4C7C3
  •	Fonts: Playfair Display (titles/headlines), Inter (body/UI)
  •	Line spacing: 120–140% (Inter body), 110–125% (Playfair headings)
  •	Logo rule: Use icon-only mark for avatars / small corners; use full lockup only when ≥120px width.
