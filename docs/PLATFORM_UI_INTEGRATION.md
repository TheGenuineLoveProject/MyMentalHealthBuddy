# Platform UI Integration — Social & Publishing

This document defines how the Social & Publishing system
appears inside The Genuine Love Project platform.

---

## UI MODULE NAME

"Reflections & Sharing"

Purpose:
Allow users to:
• Reflect privately
• Optionally share inspiration
• Never feel pressure to post

---

## UI SECTIONS

### 1️⃣ Reflection Library
• Displays approved reflection prompts
• Pulled from content pillars
• No metrics
• No engagement pressure

---

### 2️⃣ Gentle Share (Optional)
• Allows users to export a quote card
• Canva-based templates only
• Brand colors locked
• Watermark always present

CTA text example:
“Share if it feels supportive.”

---

### 3️⃣ Transparency Note (Required)

Displayed under sharing tools:

“This platform prioritizes reflection over performance.
Sharing is optional and never required.”

---

## WHAT IS NOT INCLUDED

❌ No follower counts  
❌ No likes inside app  
❌ No virality mechanics  
❌ No comparison  

---

## BRAND & SAFETY ENFORCEMENT

• Colors from colors.md only  
• Fonts from typography.md only  
• Templates from /marketing/canva only  
• No freeform publishing  

---

## RESULT

The platform supports expression
without turning users into content machines.
