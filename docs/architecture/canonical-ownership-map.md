# Canonical Ownership Map

## Runtime
Owner: PLATFORM_DOMAIN
Controls:
- server entrypoint
- port binding
- health/ready routes
- build/start commands
- deployment gate

## Healing Experience
Owner: HEALING_DOMAIN
Controls:
- journals
- reflections
- mood tools
- grounding tools
- emotional education
- crisis routing

## Business + Revenue
Owner: BUSINESS_DOMAIN
Controls:
- pricing
- checkout
- subscription surfaces
- publishing
- SEO
- analytics dashboards

Boundary:
Business may not directly influence healing flows.

## Visual System
Owner: DESIGN_DOMAIN
Controls:
- design tokens
- Lumi avatar
- motion
- contrast
- button visibility
- page coherence

## Evidence + Claims
Owner: RESEARCH_DOMAIN
Controls:
- evidence language
- uncertainty
- claim strength
- citations where needed
