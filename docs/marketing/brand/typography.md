# Typography System — The Genuine Love Project (LOCKED)

This document defines the ONLY approved typography system
for The Genuine Love Project across:
• Platform UI
• Social media
• Canva templates
• Marketing materials
• Documentation

No substitutions unless explicitly approved.

---

## PRIMARY TYPEFACES

### 1️⃣ Playfair Display
**Role:** Emotional, elegant headings  
**Usage:**
• Logo wordmark
• Quote cards
• Section headers
• Hero statements

**Styles Allowed:**
• Regular
• Bold

**Do NOT use:**
• Italics excessively
• All caps for long text

---

### 2️⃣ Inter
**Role:** Clean, readable body text  
**Usage:**
• Body copy
• UI labels
• Captions
• Subtext
• Buttons

**Styles Allowed:**
• Regular
• Medium
• SemiBold

---

## OPTIONAL ALTERNATES (CANVA FALLBACK ONLY)

Used only if Playfair or Inter are unavailable:

• Merriweather → replaces Playfair Display
• Poppins → replaces Inter

Fallbacks must match weight and spacing as closely as possible.

---

## TYPE SCALE (RECOMMENDED)

### Headings
• H1: Playfair Display — 48–64 px
• H2: Playfair Display — 36–44 px
• H3: Playfair Display — 28–32 px

### Body
• Body Large: Inter — 18–20 px
• Body Regular: Inter — 16 px
• Small Text: Inter — 14 px

---

## SPACING & READABILITY RULES

• Line height: 1.4–1.6
• Paragraph spacing preferred over line breaks
• Never compress text tightly
• White space is part of the design

---

## ACCESSIBILITY

• Minimum contrast ratio WCAG AA
• No thin fonts on light backgrounds
• Avoid long blocks of text

---

## ENFORCEMENT

This typography system is:
✔ Mandatory  
✔ Platform-wide  
✔ Non-negotiable  

Typography must reinforce calm, clarity, and emotional safety.
