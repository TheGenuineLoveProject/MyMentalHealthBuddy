# Phase 101 — Browser Startup and Service Worker Regression Verification

## Objective

Verify that the Phase 100 service-worker stale-shell fix works under browser-like startup conditions.

## Gates

The browser smoke test verifies:

- `#root` exists.
- React mounts into `#root`.
- The crisis-only fallback is not shown on normal load.
- The offline-only screen is not shown on normal load.
- “Installing” text is not visible on normal load.
- JS/CSS asset requests do not fail.
- Service worker contains the Phase 100 cache version.
- Service worker uses network-first navigation.
- Service worker uses `cache: "no-store"` for navigation.
- Service worker does not precache `/` or `index.html`.

## Boundary

No UI redesign.
No route changes.
No database changes.
No duplicate cleanup.
No avatar changes.
No content rewrites.
